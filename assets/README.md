# ClearHead Assets

This directory contains app assets including fonts and images.

## Fonts Required

Download and place the following fonts in `assets/fonts/`:

### Space Grotesk (Thinking mode)
- SpaceGrotesk-Regular.ttf
- SpaceGrotesk-Medium.ttf
- SpaceGrotesk-Bold.ttf

### Inter (Learning mode)
- Inter-Regular.ttf
- Inter-Medium.ttf
- Inter-Bold.ttf

### Orbitron (Brutal insights)
- Orbitron-Regular.ttf
- Orbitron-Bold.ttf

### JetBrains Mono (Code/mono)
- JetBrainsMono-Regular.ttf

### Satoshi (Planner) - Optional
If not available, Inter will be used as fallback.

## Download Links

- [Satoshi](https://www.fontshare.com/fonts/satoshi)

## Images

Place the following in `assets/images/`:
- `icon.png` - App icon (1024x1024)
- `splash-icon.png` - Splash screen icon
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon
